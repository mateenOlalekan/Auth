# Auth Monorepo (Client + Server)

This repository is now separated into:

- `client/` → React + Vite frontend.
- `server/` → Node.js + Express + MongoDB backend for eCommerce.

## Quick start

### Client
```bash
cd client
npm install
npm run dev
```

### Server
```bash
cd server
npm install
cp .env.example .env
npm run dev
```
